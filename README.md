# FlanaBot
A simple discord bot.

## Built with
1. [Python](https://www.python.org/) - Programming Language.
2. [Discord.py](https://github.com/Rapptz/discord.py) - API wrapper for Discord written in Python.

## Installation
1. Install Python.
2. Clone the repo.
    ```
    git clone https://github.com/AlberLC/flanabot.git
    ```
3. Install the pip packages.
   ```
   pip install -r requirements.txt
   ```
4. Add the environment variable "TOKEN" with your discord bot token.

## Run the bot
Run main.py

## Discord usage
1. Delete the last `n` messages with the command `!clear n`.
2. Mention the bot and it will respond in a rude way.
3. Sometimes the bot will reply to people without needing to mention.
